Awooga Navigator

For each host, you can define multiple regex patterns, each pattern must contain a group.

When a group is matched, the value will be extracted and used in the URL Template.

The extension will append a link to the resulting URL in the page with corresponding Link Name.

Note
  • Host URLs and patterns must be valid regexes.
  • Patterns must contain at least one capturing group.
  • URLs can contain an asterisk that will be replaced with the captured group value.
  • If there is no asterisk, the captured value is appended to the end of the URL.
